Frequently Asked Questions

Are my files in the cloud automatically backed up?

Not in the traditional sense. The cloud syncs your files, meaning if you delete a file locally, it gets deleted in the cloud too. While services have version history or a "trash" folder, this isn't a true backup. A separate backup solution is still recommended for critical data.

Is the cloud more or less secure than my personal computer?

It can be both. Major cloud providers have world-class physical and network security, likely far better than your home setup. However, the biggest risks come from user error, like weak passwords, misconfigured sharing settings, or falling for phishing attacks.
